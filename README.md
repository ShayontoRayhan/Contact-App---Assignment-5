# Contact App - Assignment 5

Quick start:

1. Install deps:

```bash
npm install
```

2. Start json-server + dev server:

```bash
npm run dev
```

This runs `json-server` at `http://localhost:3001` and the Vite dev server (usually `http://localhost:5173`).

Features implemented:
- Contacts list (fetched from `db.json`)
- Add contact page
- Show and Edit in modal
- Delete with confirmation
- Search (first/last/email/phone) and filters (First, Last, Oldest)

Files of interest: src/context/ContactContext.jsx, src/components
